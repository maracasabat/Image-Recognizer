Welcome to the Image Recognizer Project!

The Convolutional Neural Network with 96% of accuracy is applied for the image category prediction.

The main concepts used in the current neural network are:

  • Increasing Data Logging that is a strategy to significantly increase the variety of data available for training models without collecting new data.
  • Normalization, e.g., batch normalization, the method for increasing artificial neural networks' speed, performance, and stability. It normalises the input layer by re-centring and scaling the image.
  • Dropout is a regularization technique used in deep learning models to prevent overfitting, which randomly drop out (set to zero) some neurons in the network during training. This way forces the remaining neurons to learn more robust features that are generalizable to new data.
